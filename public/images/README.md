# Place exported Figma images here

Name each file `<seed>.jpg` to match the `photo(...)` seeds used in `data/*.ts`,
then set `USE_LOCAL = true` in `data/images.ts`.

Key homepage seeds: hero-factory, steps, holiday-inn, orbi-city, dar-tower, mziuri,
cat-elevator, cat-escalator, cat-travolator, news-orbi, news-otis, ...
