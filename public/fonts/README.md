# Custom font (optional — "same font as Figma")

The Figma file names **DejaVu Sans**, but it isn't a web font and lacks the uppercase
Georgian (Mtavruli) glyphs, so the site uses **Noto Sans Georgian** (full Mkhedruli +
Mtavruli support).

If you have the exact design font file, drop a **.woff2** here (e.g. `ice-display.woff2`)
and tell me — I'll add an `@font-face` and make it the primary display font, keeping
Noto Sans Georgian as the Georgian/Mtavruli fallback.
